Adria Style Studio - File Structure
====================================

Root Directory:
├── index.html              (Redirect page)
├── landing.html            (Main splash/contact page)
├── landing.css             (Landing page styles)
├── landing.js              (Landing page scripts)
├── matcher.html            (Outfit matcher demo)
├── styles.css              (Matcher page styles)
├── matcher.js              (Matcher page scripts)
├── README.md               (Repository readme)
├── PROJECT_STRUCTURE.md    (Detailed documentation)
└── clothing examples/      (Demo clothing images)
    ├── tops/
    │   ├── shopping.webp
    │   ├── shopping (1).webp
    │   ├── shopping (2).webp
    │   ├── shopping (3).webp
    │   └── shopping (4).webp
    └── bottoms/
        ├── images.jpeg
        ├── images (1).jpeg
        ├── images (2).jpeg
        ├── images (3).jpeg
        └── images (4).jpeg

Page Flow:
==========
index.html → landing.html ↔ matcher.html

Features:
=========
✓ Separated HTML, CSS, and JavaScript
✓ Two distinct pages (landing + demo)
✓ Local clothing images (10 items)
✓ Fully responsive design
✓ Upload functionality
✓ Interactive carousel
✓ Drag & drop support

To Deploy:
==========
1. Commit all changes: git add . && git commit -m "Updated structure"
2. Push to GitHub: git push
3. GitHub Pages will automatically deploy
