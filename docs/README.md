# Documentation Index

The `docs/` directory is now organized into focused categories for quicker navigation:

- **overview/** – Executive-level summaries, quick-start guidance, and onboarding checklists.
- **roadmap/** – Transformation plan, planning readme, and sprint tracking templates.
- **operations/** – Runbooks for environment management, database setup/migrations, CI/CD, and rollback procedures.
- **architecture/** – Guides for API documentation, RBAC implementation, and shared architectural patterns.
- **sprints/** – Sprint-by-sprint journals detailing goals, deliverables, and retrospectives (Sprint 01–07).

Refer to each subdirectory for the latest operational runbooks, planning artifacts, and sprint retrospectives.
